1) Netzwerkgrundlagen (IP, Subnetze, Gateway)

Lege das IPv4-Netz fest und dokumentiere Adressen. Beispiel: 192.168.10.0/24, Maske 255.255.255.0, Gateway 192.168.10.1.

Checkliste

  • IP-Plan erstellt (Server, Clients, Drucker).
  • Gateway erreichbar (ping 192.168.10.1).
  • DNS-Suffix festgelegt (firma.local).
Netzwerk Verantwortlich: Fiona

2) Domänenbeitritt der 4 Clients

  1. Systemeigenschaften → Computername → Ändern → Domäne (z. B. firma.local).
  2. Domänenkonto angeben → Neustart.
  3. Domänenanmeldung testen.
  • Zeit synchron halten (NTP).
  • DNS des DC verwenden.
Clients Verantwortlich: Fiona

3) Laufwerkszuordnung per GPO

  1. In GPMC: Benutzerkonfiguration → Einstellungen → Windows-Einstellungen → Laufwerkszuordnung.
  2. S: → \\server\shared, U: → \\server\users\%username%.
  3. Item-Level-Targeting verwenden (optional).

Checkliste

  • S: und U: sichtbar.
  • Lesen/Schreiben funktioniert.
  • gpresult /r korrekt.
GPO Verantwortlich: Fiona

4) Druckerfreigaben

  1. Drucker am Server einrichten & freigeben.
  2. GPO: Benutzerkonfiguration → Einstellungen → Systemsteuerungseinstellungen → Drucker.
  3. Testseite drucken.
Peripherie Verantwortlich: Fiona

5) Begriffserklärungen

GPO (Group Policy Object): Richtlinien für Benutzer/Computer in AD-Domänen.
GPO-Verteilung: Übertragung von GPOs an Clients durch den DC.
Domänencontroller (DC): Server für Authentifizierung & AD-Verzeichnis.
AD DS: Active Directory Domain Services – zentrale Verwaltung.
DNS: Namensauflösung Hostname ⇄ IP.
DHCP: Automatische Vergabe von IP-Adressen.
Konsolidieren: Systeme/Strukturen vereinheitlichen.
OU: Organizational Unit für Gruppen & GPOs.
Item-Level-Targeting: Zielgenaue GPO-Steuerung.
Gateway: Verbindungspunkt ins andere Netzwerk.
Nachschlagen Verantwortlich: Team
